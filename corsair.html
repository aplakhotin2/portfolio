Overview

Corsair iCUE is a powerful software platform that lets users customize and synchronize RGB lighting, monitor system performance, and fine-tune settings across compatible Corsair devices. It provides deep control over lighting effects, fan speeds, and hardware performance — offering a fully personalized and immersive PC experience.

I designed desktop and mobile for the Corsair iCUE app, working across Windows, macOS, and Chrome OS to deliver a consistent, high-quality experience to millions of gaming enthusiasts worldwide.

Design System

I created a unified design system for iCUE — covering color palette, typography (Open Sans), buttons, and UI components including navigation patterns specific to the desktop app. The design system served as the single source of truth across all platforms.

Userflow

I created user flows to figure out how real people would use the app — from the time they open it to the time they exit. User flows helped determine the content and functionality that must be on each page, aligning them with users' specific goals.

The core flow covered: Installer → Log in / Sign up → Home Page → Device Page → Device Settings.

Wireframes

Wireframes and prototypes visualize a user's interactions with an interface. This stage helped review design solutions with project stakeholders, test them before implementation, and quickly make necessary amendments.

Usability Testing

At this stage, our team ran usability testing to determine, analyze, and improve the design and user experience. Five representative users tested it to see how user-friendly the application is. The test was conducted over a video call, where the participants were given a task.

Key tasks tested:

  • Move multiple devices from one room to another
  • Sign up into the application

Based on findings, we improved the multi-device selection flow and added a verification confirmation screen on sign-up — giving users confidence that everything was completed correctly.

Final UI

Home Page
Desktop and mobile app
The home page gives users a full overview of their connected Corsair devices, scenes, and sensor readings — presented consistently across desktop and mobile.
Device Page
Desktop app
The device page provides deep control over individual devices — lighting effects per key zone, lighting layers, quick lighting zones, and a visual keyboard preview with real-time RGB preview.
Setting Page
Desktop app
General and device-specific settings — battery status, brightness, polling rate, power saving mode, angle snapping, and wireless pairing. Designed for clarity across multiple simultaneous connected devices.
Mobile Apps
iCUE iOS & Android
The mobile app brings iCUE control to iOS and Android — allowing users to manage lighting scenes, rooms, and device settings from their phone. Redesigned for a 25% improvement in usability scores.
